event对象

事件对象在不同的浏览器中访问的方法、其内置的属性和方法都不尽相同。比如以下代码在IE中无效： 

    document.onclick = function(evt) {
        for (var i = 0 in evt) {
            console.log(i + "=" + evt[i]);
        }
    };

IE将event当作全局变量，当不过只有当事件发生时，event对象才有内容。

    console.log(window.event); //null
    window.onload = function() {
        console.log(window.event);
    };
